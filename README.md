## ✨ What is AgriGuide Pro?

AgriGuide Pro is a free, AI-powered farming assistant built for India's 140 million farmers. It helps farmers diagnose crop diseases, check market prices, get weather forecasts, and chat with an AI agronomy expert — all in one beautiful app.

---

## 🚀 Features

| Feature | Description |
|---|---|
| 📸 **AI Crop Scanner** | Upload a photo → instant disease diagnosis + treatment plan |
| 🤖 **AI Farming Chat** | 24/7 expert assistant for soil, pests, irrigation & more |
| 📊 **Live Mandi Prices** | Real-time APMC wholesale rates for 12+ crops |
| 🌦 **Farm Weather AI** | City-specific forecasts with farming advisories |
| 🌱 **Crop Registry** | Track your crops and wholesale pricing |
| 📋 **Scan History** | Complete archive of all diagnoses |
| 🌐 **5 Languages** | English, Hindi, Marathi, Gujarati, Punjabi |
| 🌙 **Dark Mode** | Easy on eyes during night use |
| 🆓 **Free Forever** | No payments, no limits, no credit card |

## ⚡ Quick Start

```bash

git clone https://github.com/YOUR-USERNAME/agriguide-pro.git

cd agriguide-pro
open index.html
```

That's it! No npm, no build step, no complicated setup! 🎉

---

## 🔑 Setup Your Own API Key

1. Go to **console.groq.com** → Sign up free
2. Click **API Keys** → Create Key
3. Open `script.js`
4. Replace the key at the top:
```javascript
const GROQ_KEY = "your-key-here";
```
5. Deploy to Netlify — done! ✅

---

## 🌐 Deploy to Netlify (Free, 5 Minutes)

1. Fork this repo on GitHub
2. Go to [netlify.com](https://netlify.com)
3. Click **"Add new site"** → **"Import from Git"**
4. Select your forked repo
5. Click **Deploy** 🚀

Your site is live in 30 seconds!
